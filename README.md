Patient Registration Form

Project Description

This project is a simple HTML-based patient registration form designed to collect patient information in a structured way. It simulates a real-world hospital intake form.

Built With

HTML5

Features

User input validation using HTML5
Structured form using fieldsets and legends
Dropdown selection for departments
Radio buttons for gender and patient type
Checkboxes for symptoms
Textarea for additional notes

Purpose

This project was built as part of my learning journey in web development. It focuses on mastering HTML form structure and validation.

Future Improvements

Add CSS for styling
Add JavaScript for interactivity and validation
